Recent Projects

Wander

Wander is a photo app that keeps track of visited destinations and includes an interactive map of some of the most popular photos around the globe. Wander is primarily built with MongoDB, Express, Angular.js, Node, and Bootstrap.

Nu Notes

Nu Notes is an app in development that keeps track of users' music preferences and serves as a landing page to keep track of new music suggestions. Built with Express, Postgres, Materialize and JQuery.

Brush Up

"Brush Up: Your mind is a canvas". Users can sign up with Facebook oAuth and save/organize lessons from Khan Academy and other hand-picked educational resources via YouTube's API. Built with Ruby on Rails, along with Postgres, Materialize, and Cloudinary.

Pacific Warehouse

A choose your own adventure game built with jQuery. The player chooses different routes through an old, abandoned building in order to collect items.

Dinosaur Adventure

Dinosaur Adventure is a point-and-click game built during a 3-hour hackathon using React.js and jQuery.

Holiday Party

Created during an 8 hour AngularJS hackathon. Using data scraping from recipe web sites, the user can add recipes you'd like to make at a party.
